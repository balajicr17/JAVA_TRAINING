JAVA Path setup in Environment Variables:
=========================================
1) Go to search in desktop -> Search Advanced System settings
2) Go  Environment Varibales
3) Click New under System varibales -> set path like below
 1)JAVA_HOME ->C:\Program Files\Java\jdk-20
4) Click path add new and add this path->C:\Program Files\Java\jdk-20\bin
5) Apply -OK
